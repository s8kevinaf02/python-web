About Us

This program is completely hands-on oriented and designed to help students become Linux Systems Administrator, DevOps Engineer and AWS Admin. It will provide you with an in-depth knowledge of Linux/Unix concepts, Amazon web services (AWS), software development life cycle (SDLC), Infrastructure as Code (IaC) with Terraform and Terragrunt, bash scripting, Python scripting, and various DevOps tools including Git, Git-crypt, Bitbucket, Jenkins, Maven, Docker, Helm, Docker-Swarm, Ansible, Kubernetes or K8s, Valero, Grafana, Prometheus, Splunk and etc.
